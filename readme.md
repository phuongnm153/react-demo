## Installation

Install `laravel`:
```
$ composer install
```
Install `npm`:
```
$ npm install
```
Install `migration`:
```
$ php artisan migrate
```
Generate `data-seeding`:
```
$ php artisan db:seed
```

## Run
Open 2 command windows

Run watch compile js (Laravel mix):
```
$ npm run watch
```
Run serve php:
```
$ php artisan serve --port 8001
```
Open your browser and go to [http://localhost:8001/](http://localhost:8001/)
